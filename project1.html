Important Concepts - Webpages, Documents and Structures

Lesson 1: The Basics of the Web and HTML

Components of the Web

A PC Browser makes HTTP Requests via the internet to Hosting Servers, which respond with html files that the browser displays. Please check this video for more explanation.

HTML

HTML(HyperText Markup Language) consists of text, markup, and references to other documents such as videos and images and links to other documents.

HTML Tags and Elements

HTML markup consists of elements such as tags and contents, which are surrounded by tags. Most tags have opening and closing tags, but some tags, called void tags, only have opening tags. Attributes are the properties of elements.For example, Anchor tags use attributes that equal value such as URL that link to some documents.

Inline vs Block Elements

HTML elements are either inline e.g. br, img, b, em, span, or block e.g. p, div. block makes an invisible box around the text.

Lesson 2: Creating a Structured Document with HTML

Tree Like Structure

Different HTML elements are nested in one another. In a browser this structure looks like a series of boxes inside of other boxes. While HTML controls the structure of a web page CSS controls the style of the page.

Boxifying a Webpage

Every element on the web is contained in an invisible box. The more indented an element is when viewed as text, the more nested its box is.

Lesson 3: Adding CSS Style to HTML Structure

CSS

CSS is Cascading Style Sheets, a language that controls the style of HTML elements.

How does CSS work - in sync with HTML containers and class

CSS identifies a tag or a class of the element to match and applies rules that determine the design of the corresponding element. One benefit of using HTML containers and class is that it enables one to aviod the repetition of the same codes. HTML containers and class, along with CSS, provide similar styles and functionality to related html content, thereby helping one to write a more structured html script. 

            
              div {
              color: red;
              }

div is a selector that tells the browser that the rule color: red will be applied to every div element. color is an attribute and a red is its value. If this rule is meant to be applied to only certain divs, the corresponding HTML and CSS can be rewritten using class selectors as follows. 

            
              <div class="description"></div>
                <style>
                .description {
                  color: red;
                }
                </style>

Cascading means rules are applied not only to the elements that they directly match but also to the child elements of those elements. if a child element has overlapping rules applied to it, more specific rules take precedence. Styles are applied in the following order, with definitions further down the list overwriting previous definitions: the default style of a browser, stylesheet in a separate file, stylesheet inside HTML, inline style in an element.

How CSS can be incorporated into the webpage

1 inline style in an element

This method is not recommended because not only it leads to redundant codes but also the style attributes of every single HTML element must be modified. 

            <body>
                <div style="color: red">
                    This div will have a red text.
                </div>
            </body>

2 Stylesheet inside HTML

This method is good for small projects. CSS can be added inside a <style> element, which in turn is added inside a <head> element. 

            
             <head>
                <style>
                  div {
                  color : red;
                  }
                </style>
              </head>
              <body>
               <div>
                 This will have a red text.
               </div>
              </body>

3 stylesheet in a separate file

This method produces the most well organized web structure. All HTML can be written in one file( main.html), and all CSS code in a separate file (style.css) in the same folder. Add a <head> element to the top of the HTML and a <link> tag, along with the following attributes, inside the<head> element. <link>: *rel="stylesheet"*type="text/css"*href="style.css" 

            
              <head>
                 <link rel="stylesheet" type="text/css" href="style.css">
              </head>
              <body>
                <div>
                  This will have a red text.
                </div>
              </body>

and style.css can be written like this. 

            
              div {
              color: red;
            }
Box Sizing

HTML elements are boxes and each box has 4 components(margin, border, padding, content). To ensure correct sizing, one can set size in terms of percents rather than pixels, or set the box-sizing attribute to border-box for every element. 

            
              * {
               box-sizing: border-box;
               }
Box Positioning

Divs are block elements (as opposed to inline), so by default they take up the entire width of a page. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.

Important Concepts - Telling Computers What To Do

Lesson 1: Introduction to "Serious" Programming

Computer

Unlike most machines that are built to perform particular functions, computers can be programmed to do any desired task according to a clear sequence of instructions.

Computer Programming

A program consists of a sequence of specfic instructions that a computer follows to carry out its intended work.

Programming Language

A programming language is written in a particular language that a computer can understand and execute. It has its own grammar that's very exact about correct and incorrect expressions. Computers are not capable of contextual interpretations, so the written code has to be in exact keeping with the rules of that language's interpreter.

Python
Python is a programming language, that comes with its own interpreter that converts any written code into the instructions that a computer can understand and execute. A valid Python statement is called a Python expression. For example, 
print 1 + 1
is a correct expression, whereas 
print 1 +
isn't.

Lesson 2: Variables and Strings

Python Variables
Variables are names to which particular values are assigned. For example, in the following expression 
car = 100
, car is a variable and its value is 100. When the value is a string, it has to be surrounded by ' ' or " ". 
car = "red"
. The same variables can assume different values.
Different meanings of the '=' sign

In 1 + 2 = 3, '=' means equal. In car = 100, '=' means assignment.

How useful can variables be?
  • Variable names that explain the context of the code can enhance code readability.
  • Variables save important values.
  • Variables make it easy to change the value of something. years = years + 1
1 + 1 vs. "1"+"1"

1 is a numeric, but "1" is a string. While 1 + 1 results in 2, "1" + "1" results in "11".

Lesson 3: Input -> Function -> Output

Python Functions

A function(procedure)takes input, processes it and produces output.

Making vs. Using a function

To make a function. def name(parameter). Inside the function, write in a code that uses the input parameters. 

            
              def add(x, y):
              answer= x + y
              return answer

To use a function, call it with certain values for the parameter. 

            
              print add(1, 3)
              >>> 4
How do functions help avoid repetition

Once defined, the function can perform the intended work unless modified. This saves a great deal of time and efforts on the part of programmers

What does a return statement do

The return keyword produces the outcome of the function. It absense leads to the outcome "None".

While Loop

A while statement executes its block code any number of times as long as its test expression remains True, whereas an if statement executes its block only once if its test expression is True.

              <Block>

              if <TestExpression>:
              <Block>

Structured Data
String
  • Elements are characters.
  • Surrounded by single or doublequotes.s='yabba!'
  • Select elements using <string>[<number>].s[0]->'y'
  • Select sub-sequence using <string>[<number>:<number>].s[2:4]->bb

List
  • Elements may be any Python value.
  • Surrounded by square brackets.p=['y','a','b','b','a','!']
  • Select elements using <list>[<number>].p[0]->'y'
  • Select sub-sequence using <list>[<number>:<number>].p[2:4]->['b','b']
  • Mutable. p[0]='b'. Changes the value of the first element of p
Nestled Lists

Elements of a list can be any type, including another list. An inner element of the nestled list can be accessed through double indexing. 

            
              beatles =[['John', 1940], ['Paul', 1942], ['George', 1943], ['Ringo', 1940]]

              print beatles[1][0]
              John
Aliasing

When there are multiple names that refer to the same object, any change in that object will affect all the variables referring to it. 

            
              p = ['H', 'e', 'l', 'l', 'o']
              q = p
              q[4] = '!'
              print p
              ['H', 'e', 'l', 'l', '!']
List Operations
  • Append: adds a new element to the end of a list. <list>.append(<element>)
  • Concatenation: the + operator combines two lists without mutating either. <list> + <list> -> <list>
  • Length: the len operator can be used to find out the length of an object, which can be a list, a string, or a collection of lists and strings.len(<list>) -> <number>
